# OctoAcme Project Management Docs README

## Overview
This README provides a summary of OctoAcme's approach to project management and links to all process documentation stored in the `docs/` folder. It enables quick onboarding, clarity on workflows, and easy navigation to key guides.

## Project Management Process Summary

OctoAcme follows a **customer-first, iterative, and evidence-driven approach** to project management. This methodology ensures that every project is aligned with customer needs, continuously improved through feedback, and grounded in data-driven decisions.

### Core Roles
The following roles are essential to OctoAcme's project management framework:
- **Project Manager**: Oversees project execution, timeline, and resource allocation
- **Product Manager**: Defines product vision, requirements, and prioritization
- **Developer**: Implements features and contributes to technical decisions
- **QA**: Ensures quality standards and manages testing processes
- **Stakeholder**: Provides business context, approvals, and strategic guidance

### Key Project Stages
OctoAcme projects progress through six key stages:

1. **Initiation**: Define project scope, objectives, and stakeholders
2. **Planning**: Create detailed project plans, timelines, and resource allocation
3. **Execution & Tracking**: Implement work while monitoring progress against plans
4. **Release & Deployment**: Prepare, test, and deploy deliverables to production
5. **Risk & Communication**: Identify risks early and maintain transparent stakeholder communication
6. **Retrospective & Continuous Improvement**: Reflect on outcomes and implement process improvements

### Major Tools & Practices
- **Project Boards**: Track work items and progress visually
- **Backlogs**: Maintain prioritized lists of work to be completed
- **Stakeholder Updates**: Regular communication on status and decisions
- **Checklists**: Ensure consistency and completeness across deliverables

**Executing a project?** Reference the [Execution & Tracking](octoacme-execution-and-tracking.md) and [Risk Management & Communication](octoacme-risks-and-communication.md) guides.
